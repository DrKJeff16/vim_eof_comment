# vim_eof_comment

Ensure Vim EOF comments in given files.

## Install

```sh
pip install vim_eof_comment
```

## Usage

```sh
vim-eof-comments [-h] -e EXT1[,EXT2[,EXT3[,...]]] /path/to/dir1 [/path/to/dir2 [...]]
```
